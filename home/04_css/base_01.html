Форматы файлов

.DOC или .doc (от англ. document) - расширение имени файла, используемое для файлов, представляющие текст с разметкой или без. Расширение .DOC часто использовалось для обозначения простых текстовых файлов без формирования, поднако позже стало использоваться для ввоичных форматов с разметкой.
Документ MS Word
Portable Document Format(PDF) - межплатформенный формат электронных документов, разработанный фирмой Adobe Systems с использованием ряда возможностей языка Postscript. Документ PDF
ZIP - популярный формат сжания данных и архивации файлов. Файл в этом формате обычно имеет расширение .zip и хранит в сжаном или несжатом виде один или несколько файлов, котороео можно из него извлечь путем прспаковки с помощью спещиальной программы. Архив
